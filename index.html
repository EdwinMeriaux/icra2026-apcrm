About this guide.

This is a curated catalog of every workshop and tutorial at ICRA 2026 that's relevant to members of the IEEE RAS Technical Committee on Algorithms for Planning and Control of Robot Motion. Each session was sourced from the host workshop's own program page and ranked by relevance to the TC's scope.

How to read it.

The Monday and Friday tabs give you a chronological timeline. Click any talk card to expand and see the rationale for why it matters to the TC. The Workshops tab is the same content viewed from the other axis — by workshop rather than by time. The Speakers tab is an alphabetical index for hunting specific people.

Bullseye — directly the TC's core scope. Don't miss.
Core — strongly TC-relevant.
Adjacent — relevant for some TC subfields.

Caveats.

A few rows are marked title TBD where the speaker is confirmed but the abstract isn't yet published — typically these are inferred from the speaker's research area. Workshops where the schedule shows time blocks but doesn't map names to slots (LHMP, Multi-Agent, Space Robotics, Robots Meet Prior Maps) appear in the Workshops tab but not in the Monday/Friday timelines.

The data.

Everything in this guide is plain JSON inside this HTML file. To edit a row — fix a time, add a talk, change relevance — open this file in a text editor, find the SESSIONS or WORKSHOPS array near the top of the script block at the bottom, edit the value, save, refresh.

Print.

This page is set up to print all four content tabs as separate sections — useful if someone wants a paper handout. Use your browser's "Save as PDF" or "Print" function.
