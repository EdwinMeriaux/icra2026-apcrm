# ICRA 2026 · TC-APCRM Workshop Guide

A curated workshop guide for the **IEEE RAS Technical Committee on Algorithms for Planning and Control of Robot Motion** (TC-APCRM) at **ICRA 2026, Vienna**.

🌐 **Live site:** https://edwinmeriaux.github.io/icra2026-apcrm/

## About

This guide helps TC-APCRM members and the broader motion-planning / control community navigate ICRA 2026 — surfacing the workshops, tutorials, and sessions most relevant to the TC's scope: motion planning, control, sampling-based algorithms, optimization, learning for control, and related topics.
